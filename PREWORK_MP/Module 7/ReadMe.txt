Button Click Exercise 

https://scratch.mit.edu/projects/453320257

Rules 

three buttons, and a counter. When the up arrow is clicked the counter goes up by one. When the down arrow is clicked the counter goes down. When the middle button is clicked the counter 
resets.


if up arrow clicked 
	counter += 1
elif down arrow clicked 
	counter -= 1
else 
	counter = 0

Variable
counter

Variable will change depending on which button is clicked
